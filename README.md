# Rabotify

## Vue 2+ компоненты для Rabota.Ru

## Installing

Install dependecies

```bash
  $ yarn
```

## Dev mode (hmr support)

Run dev server at `localhost:3200`

```bash
  $ npm run dev
```

## Production build

```bash
  $ npm run build
```

## Tests

```bash
  $ npm run test
```

## Tests with ***coverage***

```bash
  $ npm run test:coverage
```
